# Employee Management System

An object-oriented Employee Management System built in Java that provides complete CRUD operations for managing employee records.

## Features

- **Employee CRUD**: Add, view, update, and delete employee records
- **Department Management**: Organize employees by departments
- **Salary Management**: Track salary details and payroll
- **Search & Filter**: Find employees by name, department, or designation
- **Data Persistence**: Store and retrieve employee data

## Technologies Used

- **Language**: Java
- **Concepts**: OOP (Inheritance, Polymorphism, Encapsulation, Abstraction)
- **Data Storage**: File-based / Database
- **UI**: Console-based / GUI

## Getting Started

### Prerequisites

- Java JDK 8 or higher
- Any Java IDE (Eclipse, IntelliJ, VS Code)

### Running

1. Clone the repository
2. Open the project in your IDE
3. Compile and run the main class
